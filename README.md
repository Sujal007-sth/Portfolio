### 3. Final Assignment Projects

#### Beckett Pizza Plaza
A pricing program implementing a "4-for-3" promotional offer.

- **Key Features:** Input validation, identification of the cheapest pizza, and final cost calculation.  
- **Concepts Used:** Lists, numeric validation, and mathematical operations.

#### Password Security Check
A password verification program simulating partial-password authentication.

- **Key Features:** Password length validation and character checks using random indexing.  
- **Concepts Used:** String manipulation, random module, and program flow control.
