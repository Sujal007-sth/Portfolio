# Introduction to Programming - Projects

This folder contains two projects for the Introduction to Programming course.

---

## Task 1 - Pizza Calculator

**Description:**  
Calculates the total cost of a pizza order for Beckett Pizza Plaza with a "Four-for-Three" offer.

**Rules Applied:**  
- Accepts four pizza prices.  
- The cheapest pizza is free.  
- Calculates total cost and discount.  
- Handles invalid inputs.

---

## Task 2 - Password Checker

**Description:**  
Verifies a user password by length and specific character checks.

**Rules Applied:**  
- Password must be at least 9 characters long.  
- Prompts for three random characters from the password for verification.  
- Exits immediately if conditions are not met.
